How to Resize pictures using imresizer.com

  • Step 1: Upload a photo.
  • Step 2: You will be provided 5 resize options.
    1. Keep Aspect Ratio - It's a checkbox which force you to preserve the height and width ratio.
    2. Unit - Choose between pixel, mm, cm, inch.
    3. DPI(dots per inch) - You can set specific DPI using this. Normally 300dpi is used for most cases, which is set by default. So you can just ignore it if you don't care about DPI.
    4. Height - Set height in your selected unit
    5. Width - Set width in your selected unit
      6. Set as many parametes as you want and apply resize.
      NOTE: You can also apply any available operation(rotate, flip, crop, resize, convert, combine) as many time as you want.
  • Step 3: Download the final photo.
